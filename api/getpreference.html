dusk.getPreference()

Description:

  • Retrieves the value of a library preference. See the preference guide for information on preference values.

Syntax:

  • dusk.getPreference(preferenceName)

Arguments:


  • preferenceName (string, required) : The name of the preference to retrieve. Names are case-sensitive.

Returns:


  • preference (preference value) : The retrieved value of the corresponding preferences/

Example:


local virtualObjectsVisible = dusk.getPreference("virtualObjectsVisible")
